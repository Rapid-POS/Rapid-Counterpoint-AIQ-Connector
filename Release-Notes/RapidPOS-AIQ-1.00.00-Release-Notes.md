# Rapid POS AIQ Connector v1.00.00 Release Notes  

_Release Date: March 2026_

---

## New Functionality

The Rapid POS AIQ Connector integrates Counterpoint with Alpine IQ (AIQ), automatically syncing customer information, item data, and posted ticket history to the AIQ platform. This enables AIQ to build unified customer personas and leverage Counterpoint sales activity for targeted email and SMS marketing campaigns.

This initial release of the connector includes:

- AIQ configuration interface
- Account store mapping and location inventory group mapping
- AIQ customer record synchronization and field mapping for Customers Up
- AIQ item record synchronization and field mapping for Items Up
- AIQ field mapping documents up
- Status views for customers and items
- Document queue and item quantity-on-hand views
- **Run AIQ Connector** button for manual troubleshooting
- **Mark All AIQ Messages as Read** option
